ScanAndRest functions

Main

  • Scan Barcode

    • Optional

  • Send the scanned Barcode to a REST-Server

  • View the resut of the REST-Server

    • REST-Server Connection

    ScanAndREST uses the following REST notation

    Get http://example.com:port/Abc?Barcode=barcode

    ScanAndRESTServer Uri from the sample

    Get http://localhost:9876/Scan?Barcode=barcode

    Code

    ScanAndREST is written with Xamarin Studio on a Mac in c# with Xamarin.Forms and the additional nuget packages. The code of the program can be found on github.
